# MACD-Testing
Takes Yahoo finance data from the given ticker variable and downloads, analyses and back-tests with a given sum, utilizing 5% of the portfolio each trade.

It also utilises matplotlib to give very basic graphics detailing the stragegy and why it is a loser by itself.

It is highly recommended to use a virtual envoirnment for this code to not have any compatability issues.

Download Instructions. 

1) Download the Latest Release and open in your favorite text editor.

2) Download the libraries from the requirements.txt file. If you are reading this past late 2025 then some of these libararys have probabbly changed. make sure to remove the # in the requirements txt if you need the exact version. 

How do run the code:

Once in the correct directory of your code: simply run the python script by python Test.py or py Test.p
